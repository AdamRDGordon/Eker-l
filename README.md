# Ekeröl

Swedish landing page for Ekeröl, a session beer brand from Ekerö.

## Files

- `index.html` - page content
- `styles.css` - visual design and responsive layout
- `script.js` - mobile menu and tasting-note dialogs

## Publish with GitHub and Vercel

1. Create a new empty repository on GitHub.
2. Upload all files in this folder to the repository.
3. In Vercel, select **Add New > Project** and import the GitHub repository.
4. Leave the framework preset as **Other** and deploy.
5. Add the custom domain `xn--ekerl-mua.se` in the Vercel project's domain settings.

No build command or environment variables are required.
